# Атлас пород — справочник о больших собаках

Учебный, но профессионально собранный справочный сайт о породах больших собак.
Демонстрирует полный цикл работы с сайтом: дизайн, вёрстка, контент, интерактив,
адаптивность, доступность и SEO.

## Что внутри

| Файл | Назначение |
|------|-----------|
| `index.html` | Главная: hero, каталог пород, методика, о проекте |
| `husky.html` | Карточка породы — эталон (Сибирский хаски) |
| `malamute.html` | Карточка породы (Аляскинский маламут) |
| `bernese.html` | Карточка породы (Бернский зенненхунд) |
| `styles.css` | Все стили (одна дизайн-система на весь сайт) |
| `script.js` | Меню, переключатель окрасов, анимации появления |

## Как открыть

Просто открой `index.html` в браузере двойным кликом. Никакой установки не нужно —
это статический сайт на чистых HTML/CSS/JS.

## Дизайн-концепция

«Иллюстрированный полевой справочник» — премиальный определитель пород.
- **Палитра:** тёплый кремовый фон (не чисто-белый), чернильно-зелёный текст,
  терракотовый акцент, мягкий мох.
- **Шрифты:** Fraunces (заголовки) + Inter (текст).
- **Сигнатурный элемент:** интерактивный переключатель окрасов, который реально
  перекрашивает иллюстрацию собаки.

## Почему это «работает эффективно» (что показать работодателю)

**Конверсия и удобство**
- Чёткая структура: на каждой странице понятно, что это и что делать дальше.
- Все породы описаны по одной схеме из 6 разделов → их легко сравнивать.
- Микровзаимодействия (переключение окрасов, hover-эффекты, плавные появления).
- Честный контент с разделом «кому подходит / не подходит» — строит доверие.

**SEO (поисковая оптимизация)**
- Уникальные `<title>` и `<meta description>` на каждой странице.
- Семантическая разметка: один `<h1>`, логичная иерархия заголовков.
- Структурированные данные Schema.org (Article + FAQPage) — помогает попасть
  в Google AI Overviews и расширенные сниппеты.
- Open Graph теги для красивых ссылок в соцсетях.
- Внутренняя перелинковка между породами.
- Осмысленный, экспертный текст (Google ценит E-E-A-T, а не «воду»).

**Производительность**
- Нет тяжёлых библиотек и фреймворков — быстрый старт (хорошо для LCP).
- Иллюстрации собак нарисованы на чистом CSS — нечего загружать.
- `preconnect` к шрифтам, `display=swap` — текст виден сразу.

**Доступность**
- Skip-link, видимый фокус с клавиатуры, ARIA-атрибуты на меню и кнопках.
- Уважение к настройке «уменьшить движение» (prefers-reduced-motion).
- Контрастный текст, читаемые размеры шрифта.

**Адаптивность**
- Корректно работает от мобильного (390px) до десктопа.
- Мобильное меню-гамбургер, перестроение сеток.

## Как масштабировать (добавить новую породу)

Архитектура специально сделана так, чтобы новая порода не ломала дизайн:
1. Скопируй `husky.html` → переименуй (например `labrador.html`).
2. Замени контент: title, meta, заголовок, факты, 6 разделов, окрасы.
3. Добавь карточку в каталог в `index.html`.
4. Поправь ссылки «предыдущая/следующая» внизу страниц.

Это и есть профессиональный подход: 3 готовые породы доказывают, что система
масштабируется на любое количество.

## Как добавить настоящие фото

В разделе «… в жизни» стоят блоки-плейсхолдеры. Замени каждый
`<div class="photo-ph">…</div>` на `<img src="фото.jpg" alt="описание" loading="lazy">`.
Подписи и сетка уже готовы. Атрибут `loading="lazy"` сохранит скорость загрузки.
